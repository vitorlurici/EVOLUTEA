🚀 EvoluTEA

EvoluTEA é uma plataforma dedicada ao desenvolvimento de crianças com Transtorno do Espectro Autista (TEA), oferecendo suporte por meio de conteúdos estruturados, orientações práticas e recursos acessíveis para famílias e profissionais.

Nosso objetivo é promover mais autonomia, comunicação e qualidade de vida, respeitando o tempo e as necessidades individuais de cada criança.

🧠 Sobre o Projeto

A plataforma foi desenvolvida com foco em acessibilidade, usabilidade e impacto social, permitindo:

Acompanhamento do desenvolvimento infantil
Acesso a conteúdos educativos
Orientações para pais e profissionais
Interface simples e intuitiva

⚙️ Como rodar o projeto
🔹 Pré-requisitos
Node.js instalado
Java (JDK 17+)
PostgreSQL configurado
🔹 Frontend (Next.js)
# Clone o repositório
git clone https://github.com/seu-usuario/evolutea.git

# Acesse a pasta do frontend
cd evolutea/frontend

# Instale as dependências
npm install

# Rode o projeto
npm run dev

🔹 Backend (Spring Boot)
# Acesse a pasta do backend
cd evolutea/backend

# Rode a aplicação
./mvnw spring-boot:run

🔹 Banco de Dados (PostgreSQL)
Crie um banco de dados:
CREATE DATABASE evolutea;

Configure as credenciais no application.properties:
spring.datasource.url=jdbc:postgresql://localhost:5432/evolutea
spring.datasource.username=seu_usuario
spring.datasource.password=sua_senha
